# 交易后台管理系统

这是一个基于 Vue 3 + Vite + Element Plus 构建的后台管理系统，具有完整的业务单和交易单管理功能。

## 功能特性

- 响应式布局，支持左侧菜单导航
- 业务单管理（增删改查功能）
- 交易单管理（增删改查功能）
- 支持按条件搜索过滤数据
- 分页展示大量数据
- 模态框表单操作

## 技术栈

- Vue 3 (Composition API)
- Vite (构建工具)
- Element Plus (UI组件库)
- Vue Router 4 (路由管理)
- Pinia (状态管理)

## 安装与运行

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 访问应用：
   打开浏览器访问 http://localhost:5174/

## 项目结构

```
src/
├── assets/           # 静态资源
├── components/       # 公共组件
│   └── Common/       # 通用组件
├── views/            # 页面视图
├── router/           # 路由配置
├── Layout.vue        # 主布局组件
└── main.js           # 应用入口文件
```

## 功能模块

### 业务单管理
- 单号、状态、创建时间、操作人、备注等字段展示
- 支持按单号、状态搜索
- 新增、编辑、删除功能

### 交易单管理
- 订单号、用户、金额、支付状态、创建时间等字段展示
- 支持按订单号、支付状态搜索
- 新增、编辑、删除功能
